# automl-llm

AutoML LLM 项目仓库

## 目录结构
- app/         应用相关代码
- core/        核心模块
- prompts/     提示词与模板
- examples/    示例与用例
- artifacts/   产物与模型文件

## 简介
AutoML-LLM 是一个结合自动化机器学习（AutoML）与大语言模型（LLM）的数据分析平台。项目旨在通过智能化的数据处理、特征工程、模型选择与评估，提升数据分析效率，并利用 LLM 实现自然语言交互与自动化决策建议。适用于科研、企业数据分析、教学等多种场景。

## 使用方法

### 步骤一：创建并激活虚拟环境
建议在项目根目录下创建并激活 Python 虚拟环境，以便管理依赖。

#### 激活虚拟环境
```powershell
.\venv\Scripts\activate
```

#### 安装依赖
```powershell
pip install -r requirements.txt
```
